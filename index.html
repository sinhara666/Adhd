⚡ Quick Setup (3 steps)

  1. 1 Get a free Groq API key at console.groq.com — sign up, go to API Keys, create one. It's free.
  2. 2 Paste your key in the Groq Key field on the Today tab and hit Save. It stays in your browser — never sent anywhere except Groq.
  3. 3 Add this file to your phone's home screen: in your browser tap Share → Add to Home Screen. Opens like an app, no install needed.

📌 Tips

  1. →Your plan is cached — reopen the app anytime today and it loads instantly without burning an API call.
  2. →Check off tasks as you do them. Progress is saved in your browser.
  3. →If you derail, hit the red reset button — it generates a refocus plan for where you are right now.
  4. →Your task brain dump saves automatically so it's there next time you open.

🔑 Privacy

  1. →Your Groq key and all plan data lives in your browser's localStorage only. This tool has no server, no account, no tracking.
